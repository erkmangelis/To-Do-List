React ile yaptığım To-Do List projesi.

Özellikler:
- Yeni görev oluşturabilme.
- Var olan görevi silme.
- Görevleri sürükle-bırak ile sırasını değiştirebilme.
- Görevleri sürükle-bırak ile bulunduğu kategoriyi değiştirme.
